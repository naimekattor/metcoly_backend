# Immigration Consultation & Case Management Platform

A comprehensive backend platform for immigration consultation and case management with role-based access control, payment processing, document management, and analytics.

## 🚀 Features

- **Multi-role System**: SUPER_ADMIN, CONSULTANT, CLIENT
- **Authentication**: JWT-based authentication with refresh tokens
- **Application Management**: Complete workflow from draft to approval
- **Booking System**: Consultation booking with Calendly integration
- **Document Management**: Version-controlled document uploads
- **Payment Processing**: Stripe integration for consultation and processing fees
- **Analytics Dashboard**: Comprehensive metrics and reporting
- **Audit Logging**: Complete activity tracking for compliance

## 📋 Prerequisites

- Node.js (v18 or higher)
- PostgreSQL (v14 or higher)
- npm or yarn
- Stripe account (for payments)
- SMTP server (for emails)
